Debug Example

This page simply demonstrates how to use debug instead of console to insure there are no javascript errors if the browser doesn't support the console object

The debug feature can be turned on or off. There are two ways to do this.

  1. Set the debug.on property to "true". The default is "false".
  2. Add a query string to the URL of debug=true, http://example.com?debug=true

If the browser doesn't have a console, all debug information will be added to debug.history.

If you are using a browser which supports the console object, just view the console and you'll some output from the debug object.
